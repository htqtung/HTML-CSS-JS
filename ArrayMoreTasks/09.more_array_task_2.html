Blood Type

Write a program that allows the user to enter a blood type. When the user clicks the "Submit" button, the program shows the percentage of the blood type found in the following array:

var bloodTypes = ["A+", "O-", "AB+", "O+", "AB+", "AB+", "O-", "AB+", "O+", "AB+"];

For example, if the user enters "AB+" as the blood type, the program will print "AB+ 50 percent" as the answer.
